# Guia de Estilo e Cores da Tecla T

Este documento detalha a paleta de cores e a tipografia utilizadas no site da Tecla T, com base na análise da identidade visual da empresa.

## Paleta de Cores

A Tecla T utiliza uma paleta de cores moderna e profissional, com destaque para tons de verde/azul vibrantes que transmitem inovação e tecnologia, combinados com cores neutras para o fundo e texto, garantindo legibilidade e uma estética clean.

| Nome da Cor | Código Hexadecimal | Código RGB | Uso Principal |
|---|---|---|---|
| **Verde/Azul Vibrante** | `#00C896` | `rgb(0, 200, 150)` | Destaques, botões primários, elementos interativos |
| **Azul Escuro (Fundo)** | `#1A1A2E` | `rgb(26, 26, 46)` | Fundo principal do site |
| **Branco (Texto)** | `#FFFFFF` | `rgb(255, 255, 255)` | Texto principal, títulos em fundos escuros |
| **Cinza Claro (Texto)** | `#E0E0E0` | `rgb(224, 224, 224)` | Texto secundário, detalhes |
| **Azul (Links)** | `#007BFF` | `rgb(0, 123, 255)` | Links e elementos de interação secundários |
| **Transparente** | `transparent` | `rgba(0, 0, 0, 0)` | Elementos com fundo transparente (ex: ícones, bordas) |

## Tipografia

A Tecla T emprega fontes modernas e legíveis para garantir uma experiência de usuário agradável e profissional. A principal família de fontes identificada é a **Inter**, uma fonte sans-serif conhecida por sua clareza e versatilidade em diferentes tamanhos e contextos digitais.

| Família de Fonte | Exemplos de Uso |
|---|---|
| **Inter** | Títulos, corpo de texto, botões |
| **sans-serif** | Fallback genérico para garantir compatibilidade |
| **ui-sans-serif, system-ui, -apple-system, BlinkMacSystemFont, Segoe UI, Roboto, Helvetica Neue, Arial, Noto Sans, sans-serif, Apple Color Emoji, Segoe UI Emoji, Segoe UI Symbol, Noto Color Emoji** | Stack de fontes para garantir renderização consistente em diferentes sistemas operacionais e navegadores |

## Estilização Geral

O estilo geral do site da Tecla T é **minimalista e focado na usabilidade**, com:

*   **Gradientes sutis:** Embora não explicitamente capturados pelo script, o site utiliza gradientes em alguns elementos, especialmente em botões e seções de destaque, para adicionar profundidade e modernidade.
*   **Animações fluidas:** Transições suaves e animações discretas contribuem para uma experiência interativa e profissional.
*   **Fundo escuro:** O uso predominante do azul escuro (`#1A1A2E`) como cor de fundo confere um ar sofisticado e destaca o conteúdo em branco e verde/azul vibrante.
*   **Bordas arredondadas:** Elementos como botões e cards podem apresentar bordas levemente arredondadas para um toque mais suave e moderno.

Este guia serve como referência para manter a consistência visual em todos os materiais e plataformas da Tecla T. [1]
